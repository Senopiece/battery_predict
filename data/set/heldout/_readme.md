Processed manual holdout battery data as JSONL cycle records.

Generated by:
`python data/raw/batterylife/convert.py`

Output format (`*.jsonl`), one line per cycle:

`{"V": [ ... ], "A": [ ... ]}`

Where:
- `V`: voltage signal samples
- `A`: current signal samples

Signals are resampled to 1 Hz during conversion, so an explicit time column is
not stored.

Each output file uses deterministic 4-character base62 naming from payload
contents, consistent with the main conversion naming strategy.
