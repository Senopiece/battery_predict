Processed battery tensors in the project format `(cycle, sample, channel)`, one `.npy` file per cell.

Files are generated by running the converters in `data/raw/*/convert.py`.

Directory layout:
- `data/set/*.npy`: training/validation pool
- `data/set/heldout/*.jsonl`: manual holdout pool with one JSON record per cycle

A pre-built version of this dataset (ready to use without running converters) is available on Kaggle:
https://www.kaggle.com/datasets/nabuki/battery-predict-dataset
